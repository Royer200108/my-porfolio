## ¿En qué consistió este proyecto?
Fue un proyecto estudiantil, realizado en conjunto con otros programadores para poder realizar el análisis y desarrollo completo de una webapp que funcionase como una plataforma para poder alquilar autos.

## Mi participación en el proyecto.
En este proyecto fungí como diseñador de bases de datos y además como programador y diseñador frontend, lo que me permitió desarrollar y mejorar mis habilidades y experiencia con bases de datos relacionales, además del uso de herramientas de desarrollo web.
