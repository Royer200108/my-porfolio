## ¿En qué consistió este proyecto?
Fue un proyecto estudiantil, realizado en conjunto con un colega para poder aplicar las conocimientos adquiridos en el desarrollo web.

## Mi participación en el proyecto.
Mi participación en este proyecto fue como desarrollador frontend y backend, tambien apoye en la creación y diseño de la base una base de datos no relacional, lo que me permitió introducirme un poco en el mundo de bases de datos no SQL. Tambien en la realización de este proyecto tuve la oportunidad de configurar un servidor web montado en ubuntu linux para poder poner en producción dicho proyecto.
