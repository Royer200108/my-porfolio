## ¿Cuál es el fin de este proyecto?
Este es un proyecto acádemico, realizado de manera personal con el fin de introducir conceptos de **Infrastructure as code** y **Computación en la nube**.

## ¿En qué consistió este proyecto?.
Este proyecto consiste en el análisis, planificación e implementación de infraestructura en la nube para una aplicación de contabilidad.

### ¿Qué contiene el grupo de recursos de este proyecto?

#### ¿Qué es un grupo de recurso de Azure?
Un grupo de recursos en Azure es un contenedor lógico que agrupa múltiples recursos relacionados, como máquinas virtuales, bases de datos y redes, para facilitar su administración, implementación y organización.

#### ¿Qué es un recurso de Azure?
Un recurso de Azure es cualquier servicio o componente que se crea y gestiona dentro de la plataforma de Microsoft Azure, como máquinas virtuales, bases de datos, redes virtuales, aplicaciones web o funciones.

Para la realización de este proyecto se incluyeron los siguientes recursos.
* App Service Plan.
* Redes virtuales (VNets).
* Sub redes (SubNets).
* Servidor de Bases de Datos (Azure SQL).
* Almacenamientos (Blob storage y Queue storage).
